# Beverage Prices

This is the Beverage Prices kata in TypeScript.

## Getting started

Install dependencies

```sh
yarn install
```

## Running tests

To run all tests

### Jest way

```sh
npm run test:jest
```

To run all tests in watch mode

```sh
npm run test:jest:watch
```

### Mocha way

```sh
npm run test:mocha
```
